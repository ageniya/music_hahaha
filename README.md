# 💒 婚礼音乐库 — BGM 管理 & 歌单制作

一站式婚礼仪式音乐管理工具。无需安装软件，打开网页即可：浏览曲库、剪辑音频、制作歌单、导出 ZIP 压缩包，发给主持人直接用。

---

## 🚀 快速开始

### 方式一：在线访问（推荐 ✅）

用手机或电脑浏览器打开：

```
https://ageniya.github.io/music_hahaha
```

### 方式二：本地打开

1. 下载项目所有文件
2. 双击 `index.html` 打开
3. 网页会提示"音频未加载"——点击工作区上方的 **📥 一键加载音频**，全选 `data/audio/` 文件夹里的 MP3 文件即可

---

## 📖 使用说明

### 1️⃣ 浏览音乐库

页面左侧是 **📚 音乐库**，列出了婚礼仪式的 12 首预设歌曲：

- 可按 **环节**（灯光秀 / 新娘入场 / 誓言…）、**歌手** 筛选
- 搜索框支持歌名、歌手、备注
- 🟢 表示音频已就绪，可双击播放

### 2️⃣ 制作歌单

把需要的歌曲加到右侧 **🛠️ 工作区**：

1. 在音乐库中点击歌曲右侧的 **+** 按钮
2. 歌曲会添加到工作区，音频自动加载
3. 工作区中的歌曲可以：
   - **✏️ 重命名** — 改成主持人容易看懂的名字
   - **✂️ 剪辑** — 打开音频剪辑器，裁剪需要的片段
   - **拖拽排序** — 按住拖拽调整出场顺序
   - **✕ 移除** — 从工作区删除

### 3️⃣ 剪辑音频

点击工作区歌曲的 ✂️ 按钮打开剪辑器：

| 操作 | 方法 |
|------|------|
| 拖拽选取 | 拖拽波形图上的 ▶ 起点和 ⏹ 终点手柄 |
| 手动输入 | 在"开始时间/结束时间"框中输入 `分:秒` |
| 预览 | 点击 **▶ 预览剪辑** 试听 |
| 保存 | 修改名称后，点 **💾 保存到工作区** |

剪辑后的音频以 WAV 格式保存，文件名自动编号。

### 4️⃣ 导出 ZIP

工作区准备好后，点击右上角 **📦 导出 ZIP**：

- 自动生成 `婚礼歌单.zip`
- 内含编号排序的音频文件（01_xxx.mp3、02_xxx.wav…）
- 附带 `歌单信息.json` 包含曲目列表

> 💡 你也可以点 **📤 导出信息** 单独导出歌单 JSON（不含音频）。

### 5️⃣ 保存 & 加载歌单

工作区底部操作栏：

- **💾 保存歌单** — 把当前工作区保存到浏览器，下次打开可恢复
- **📂 加载歌单** — 从已保存的歌单列表中恢复
- **🗑️ 清空** — 清除当前工作区

## ❓ 常见问题

**Q: 为什么双击歌曲显示"音频未加载"？**
A: 本地 `file://` 打开时浏览器禁止自动加载音频。点击工作区上方的 **📥 一键加载音频** 按钮，全选 `data/audio/` 中的 MP3 文件即可。在线访问则无此问题。

**Q: 工作区数据会丢失吗？**
A: 工作区和歌单都保存在浏览器本地存储中，清空浏览器数据会丢失。建议定期 **导出 ZIP** 备份。

**Q: 如何替换曲库中的歌曲？**
A: 点击 **📥 上传 MP3** 选择本地文件。文件名以 `a_`、`b_`…开头会自动匹配对应环节的预设歌曲。

## 🛠️ 技术栈

纯前端，无需后端服务器：

- **Web Audio API** — 音频解码、剪辑、WAV 编码
- **Canvas** — 可视化波形图
- **IndexedDB** — 音频文件本地持久化
- **JSZip** — 浏览器端 ZIP 打包
- **GitHub Pages** — 静态站点托管
